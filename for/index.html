For 循环

注意以下代码是我们第一看到缩进。 python 使用语法块缩进。相当于其他语言的{} for 的下一条是要相对for语句缩进的。 我会在if小节详细描述。

l1 = ['a','b','c']
for i in l1:
  print(i)

结果

a
b
c

第二段代码

l2 = ['2','a',1,'d']
for i in range(0,4):
   print(l2[i])
