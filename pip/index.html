安装其他模块

现在 python 将第三方模块 放在 https://pypi.org/

我们可以通过 pip 去安装 pypi.org上的库。

目前Windows 安装python以后就自带了 pip命令。 Mac OSX 也是自带了pip。 Linux需要安装一下。 例如：ubuntu ，sudo aptitude install python3-pip 不同的Linux 安装命令有差异。可能需要自己寻找一下。

使用方法

注意：：：：这是命令行里面输入的，例子为:ubuntu 系统

sudo pip install requests

安装一个requests 网络请求模块.

引用安装好的模块

这是代码里面引用的。

import requests
